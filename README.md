puppet-s3cmd
============

A puppet module to manage s3cmd.

## Sample Usage
Install s3cmd to "vagrant" user:
```puppet
node default {
	class {'s3cmd':
		user       => 'vagrant',
		access_key => 'ACCESS_KEY',
		secret_key => 'SECRET_KEY',
	}
}
```

Purge completly s3cmd:
```puppet
node default {
	class {'s3cmd':
		ensure => absent,
		user   => 'vagrant', # remove ".s3cmd" file in user's home.
	}
}
```
